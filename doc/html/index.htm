About Trading Engine

Trading Engine also known as TE, is a pragmatic implementation of a marketplace. The TE solution that comes with

  • instrument model
  • price / time matching
  • market data distribution
  • order pre-trade validation
  • crypto depository (exchange wallets) with gateways to crypto networks (Bitcoin and Ethereum).
  • GUI for montitoring and controlling the solution.


The trading engine provides a REST / WEB Socket API allowing clients to connect to the marketplace.

The solution is provided as an open source code project under the Apache 2.0 licensing. Currently, version 1.2 is released and further work is in progress. For future works see future plans

For Whom
For any one that like to setup and run a market place or further develop the solution to meet own requirements.

Deploying Trading Engine

  • TE is provided as a public GIT HUB project.
  • The project contains a pre-configuration allowing the market place to run out of the box.
  • There are scriptd for starting the components, monitoring and controlling the solution like adding users, managing order on behalf, view statistics etc..
  • The TE has been developed with simplicity and performance in mind. Typically requests executes in sub milliseconds. If you are interested in the perfomance characteristics you will find futher information here.

Crypto Gateways
The solution includes two crypto gateways allowing the TE to interact with Bitcoin and/or Etherreum. The solutions implements exchange wallets allowing client to deposit coins for trading.
